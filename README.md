<!-- GETTING STARTED -->
## Getting Started

### Prerequisites
- [Qt 6.8.0](https://www.qt.io/download-qt-installer)

### Installation
1. Install Qt 6.8.0 on your system.
2. Open the Ripple.pro file in the project directory.
3. Select the MSVC 2019 64-bit compiler.
4. Build the project as Release.
